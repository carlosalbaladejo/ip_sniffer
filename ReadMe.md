# Ip_sniffer

Following [Tensor Programming tutorial](https://www.youtube.com/watch?v=-Jp7sabBCp4&list=PLJbE2Yu2zumDD5vy2BuSHvFZU0a6RDmgb)

Use:

- Help

```bash
    ip_sniffer -h
```

- Set number of threads

```bash
    ip_sniffer -j <number of threads> <ip>
```

- Call with default config

```bash
    ip_sniffer <ip>
```
